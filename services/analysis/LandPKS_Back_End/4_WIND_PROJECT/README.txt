1. Require Python 2.7.3 or above

2. Extract Weather_Files.zip project and cop Weather_Files folder in to your workspace such as ..\Working

2. OPS files are put in ..\Working\6_OPERATIONS_PROJECT\Operation_Files\Operation_Files

3. Run script by command line
Run :   python main_OPERATION.py -f \Operation_Files\Operation_Files

If you want to delete old data file, run command :  python main_OPERATION.py -rm


4. Waiting for a moment for script running

5. After complete with main_ISRIC, file data OPSCCOM.DAT will be located in Folder : \Working\6_OPERATIONS_PROJECT\Operation_Files\DAT_Files
